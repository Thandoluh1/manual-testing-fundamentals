# Bug Reports

## Bug ID: BUG_001
- **Title:** Login button enabled when fields are empty
- **Severity:** Medium
- **Priority:** High
- **Environment:** Chrome, Windows 10

### Steps to Reproduce
1. Navigate to the login page
2. Leave username and password fields empty
3. Observe the Login button

### Expected Result
Login button should be disabled until required fields are populated.

### Actual Result
Login button remains enabled.

### Status
Open
